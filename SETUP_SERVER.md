# حل مشكلة عدم ظهور المنتجات بعد إضافتها

## المشكلة
عند إضافة منتج جديد من لوحة الإدارة، لا يظهر المنتج بعد الإضافة أو يختفي عند إعادة تحميل الصفحة.

## السبب الرئيسي
السيرفر (json-server) غير مشغل، لذا لا تُحفظ المنتجات الجديدة في قاعدة البيانات.

## الحل

### 1️⃣ شغّل السيرفر في نافذة طرفية جديدة:
```bash
npm run server
```
سيظهر الرسالة:
```
JSON Server is running on http://localhost:3001
```

### 2️⃣ في نافذة طرفية أخرى، شغّل تطبيق الـ React:
```bash
npm run dev
```

### 3️⃣ الآن يمكنك:
- الذهاب إلى لوحة الإدارة (/admin/products)
- إضافة منتجات جديدة
- ستظهر المنتجات فوراً في الموقع
- ستبقى المنتجات محفوظة حتى بعد إعادة تحميل الصفحة

## ملاحظات مهمة
- ⚠️ تأكد دائماً من تشغيل السيرفر قبل إضافة أي منتجات
- البيانات تُحفظ في ملف `db.json`
- عند حذف المنتجات، يتم حذفها نهائياً من قاعدة البيانات
- السيرفر يستخدم المنفذ 3001

## استكشاف الأخطاء

### الخطأ: "Unable to connect to the server"
**الحل**: شغّل `npm run server` في نافذة طرفية جديدة

### المنتج لا يظهر بعد الإضافة
**الحل**: تأكد من:
1. السيرفر مشغل على المنفذ 3001
2. لا توجد أخطاء في console (F12)
3. حاول تحديث الصفحة (F5)

### خطأ "EADDRINUSE"
**الحل**: المنفذ 3001 مشغول. استخدم:
```bash
npm run server -- --port 3002
```
وغيّر `API_URL` في `.env` أو `vite.config.ts`
